# node-red-contrib-glartek-collector
Glartek Collector allows to injest data into Glartek ecosystem. This is the Node-Red version.

## Dependency
- mqtt 3.0.0
- mqtt-nedb-store 0.1.1

## Installation
`npm install node-red-contrib-glartek-collector`
